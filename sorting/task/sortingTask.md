# Задача 1: Объединение отсортированных массивов

## Описание

У тебя есть два отсортированных массива `arr1` и `arr2`. Тебе нужно объединить их в один отсортированный массив без использования встроенных функций сортировки.

### Пример 1:
```
arr1 := []int{1, 3, 5, 7}
arr2 := []int{2, 4, 6, 8}

```
